# Codeforces. VKCup Hall of fame

Небольшой проект для генерации рейтинга участников VK Cup.

Несколько лет подряд (15-18 года) VK проводил соревнование по программированию на площадке Codeforces. Одним из условий к участникам было:
> ранее принимал участие в Финалах VK Cup не более одного раза (учитываются Финалы с момента проведения чемпионата в командном формате, то есть с 2015 года)

Из-за этого появилась мысль - собирать результаты финалов и генерировать рейтинг участников, находить тех, кто уже дважды попадал в финал.

## Stack

- .Net Core 3.1
- [CodeforcesApiWrapper](https://github.com/InRedikaWB/CodeforcesApiWrapper) - библиотека для работы с Codeforces API
